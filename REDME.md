# Server Performance Monitoring Script (Bash)

A lightweight Bash script that collects and summarizes Linux server performance metrics.

## Features
- CPU usage percentage
- Memory usage (used/free/available)
- Disk usage summary
- Top 5 processes by CPU
- Top 5 processes by memory
- OS release info, uptime, load average, logged-in users
- Works across common Linux distributions

## How to run
```bash
chmod +x server-stats.sh
./server-stats.sh
